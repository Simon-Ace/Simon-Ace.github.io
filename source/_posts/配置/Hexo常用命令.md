## 常用命令

- **生成静态文件**

```shell
hexo generate
# 简写
hexo g
```

- **启动服务预览文章**

```shell
hexo server
# 简写
hexo s
# 指定端口
hexo server -p 5000
```

- **一键部署**

```shell
hexo deploy
# 简写
hexo d
```
